== Installation ==

1. Upload the plugin files to `/wp-content/plugins/signalkit-for-google/` directory, or install through WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to SignalKit in the WordPress admin menu
4. Configure your Google News URLs and customize your banners
5. Enable the banners and you're done!

= Minimum Requirements =

* WordPress 5.0 or higher
* PHP 7.2 or higher
* MySQL 5.6 or higher

== Frequently Asked Questions ==

= Do I need a Google News account? =

Yes, you need to have your publication listed on Google News to use the Follow banner. For the Preferred Source banner, users need access to Google News settings.

= Can I run both banners at the same time? =

Absolutely! Both banners are independent and can be displayed simultaneously with different positions and settings.

= Will this slow down my website? =

No. SignalKit is lightweight and optimized for performance. It only loads minimal CSS and JavaScript.

= Can I customize the banner appearance? =

Yes! You have full control over colors, text, positioning, animations, and display rules for each banner.

= How do I get my Google News URL? =

Visit your publication on Google News and copy the URL. It typically looks like: https://news.google.com/publications/...

= Is this GDPR compliant? =

Yes. The plugin only uses functional cookies for banner dismissal (with user action) and doesn't collect personal data.

= Can I show banners only on specific pages? =

Yes! You can control display by page type (posts, pages, homepage, archives) and device type (mobile/desktop).
